## 📚 About CS50x

**CS50x** is Harvard's Introduction to Computer Science, designed to teach the foundations of computer science and programming. It covers a wide range of topics including:

- C Programming
- Memory Management
- Data Structures
- Algorithms
- Python
- SQL
- Web Programming
- Flask and more
